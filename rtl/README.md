

## ISA  

### Opcodes
- ADD:  0000
- SUB:  0001
- AND:  0010
- OR:   0011
- XOR:  0100 
- SLL:  0101 // shift left logical
- SRL:  0110 // shift right logical
- SRA:  0111 // shift right arithmetic
- SLT:  1000 // set less than
- SLTU: 1001 // set less than (unsigned)
- MUL:  1010 // multiply (lower 32-bits)
